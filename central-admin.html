Order Workflow

Every step from sale to retention — in order.

1
Customer buys

Buyer selects a print, pays via Paystack. Confirmation email fires automatically. You get an alert via Formspree. The COA dialog appears for the buyer to save or print.

Automatic — no action needed

2
Print & ship

Print is produced (7–14 business days). Package and ship via DHL. Then use the Ship Order tab to send the buyer their tracking number.

Action: Ship Order tab →

3
Send COA (if needed)

The COA is usually sent automatically at purchase. Use the Send COA tab only to resend or issue one manually (e.g. for a gift or a print sold in person).

Action: Send COA tab → (manual only)

4
Post-delivery follow-up

7–10 days after shipping, send a warm check-in. Ask how the print arrived, invite them to share a photo. Builds loyalty and often leads to repeat buyers.

Action: Follow-up tab →

5
Waitlist (when sold out)

When a print sells out, add interested buyers to your list. Use the Waitlist tab to notify them first when it restocks — before announcing publicly.

Action: Waitlist tab →

Order Tracker — Google Sheets Setup

Every Ship, COA, and Follow-up you send from this page can be automatically logged to a Google Sheet. One-time setup — takes about 5 minutes.

Step 1 — Create a new Google Sheet. Name it Jeffery Asare Orders.
Step 2 — Click Extensions → Apps Script.
Step 3 — Delete the default code and paste the script below.
Step 4 — Click Deploy → New Deployment → Web App. Set "Who has access" to Anyone. Click Deploy. Copy the web app URL.
Step 5 — Open central-admin.html in a text editor, find var SHEET_URL = ''; and paste your URL between the quotes. Commit and push.
